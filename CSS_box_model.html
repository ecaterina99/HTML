În lecția anterioară au fost prezentate tehnicile pentru stilizarea textului în documentul HTML. Înainte de a fi prezentate tehnici de stilizare a celorlalte elementele HTML, este necesară o familiarizare cu unele principii de bază ale stilizării CSS care influenţează stilizarea tuturor elementelor HTML. Astfel, această lecție va fi dedicată diviziunii de bază a tuturor elementelor HTML în block și inline. Vom vorbi şi despre Modelul CSS Box, care definește zonele ce înconjoară fiecare element HTML. O bună înțelegere a conceptelor care vor fi prezentate în această lecție este condiția prealabilă pentru definirea de succes a stilizării elementelor HTML.
